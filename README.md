# 本書の使い方

1. **本文をざっと読む**<br>
本書は「理論を学ぶための演習書」という立ち位置なので，本文はざっと眺める程度にとどめ，演習に取り掛かってしまいましょう．
特に「第2章 常微分方程式」「第3章 線形微分方程式」はテクニックゲーなので，この傾向が強いです．
  
2. **演習問題を解く**<br>
積極的に答えを参照しながら，解いていってください．
いくつか演習問題が解き終わって改めて本文を振り返ると，「こういうことを言いたかったのか」と納得いく場合が多いです．
